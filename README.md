# Servicio de Edición de Videos - Waldo

Página web promocional para servicios de edición de videos especializados en invitaciones de cumpleaños, eventos grandes y bodas.

## Características

- **Diseño Elegante y Formal**: Utiliza tipografía serif para un aspecto sofisticado.
- **Totalmente Responsivo**: Se adapta a cualquier dispositivo, desde móviles hasta desktops.
- **Menú Hamburguesa**: Navegación móvil intuitiva con animaciones suaves.
- **Secciones Principales**:
  - Hero: Presentación del servicio.
  - Servicios: Detalles de los tipos de eventos.
  - Galería: Ejemplos de trabajos (placeholders).
  - Contacto: Formulario para consultas.

## Tecnologías Utilizadas

- HTML5
- CSS3 (con Grid y Flexbox para layouts responsivos)
- JavaScript (ES6+)

## Cómo Usar

1. Abre `index.html` en tu navegador web.
2. Navega por las secciones usando el menú.
3. En dispositivos móviles, usa el botón hamburguesa para acceder al menú.
4. Prueba el formulario de contacto (simulado).

## Personalización

- Reemplaza las imágenes placeholder en la galería con tus propios trabajos.
- Actualiza el contenido de texto según tus servicios específicos.
- Modifica los colores en `:root` en `css/style.css` para adaptar al branding.
